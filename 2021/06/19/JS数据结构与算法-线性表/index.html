数组及链表知识点

  1. 数组的创建：const arr = (new Array(10)).fill(1);const arr = [];
  2. 数组的循环和遍历：
    1. for(let i = 0; i < arr.length; i++);//遍历最快
    2. arr.foreach((item, index) => {console.log(item, index)})
    3. arr.map(i -> i+3);//与foreach的区别在于返回新数组
    4. 二维数组创建：初始化时注意区分原始类型和引用类型。
  3. 数组的增删：
    1. 增：unshift(), push(), splice(index,0,value)
    2. 删：shift(), pop(), splice(index, 1)
  4. 栈和队列的特点：基于数组的api。
  5. 链表：
    1. 创建：构造函数。function ListNode(val){this.val = val;this.next = null}
    2. 节点的增删：挪移指针。
  6. 链表与数组的辨析：
    1. 快慢数组的区别：纯数字数组对应连续内存，不同数据类型则对应哈希值，由对象链表实现。
    2. 增删查改的时间复杂度对比。
